About

Simple URL Shortener is a really simple URL shortener. It's open source so anyone can look at the code on GitHub. This website is just a quick project i put together because it seemed like a nice challenge.

Open Source

The best thing about Simple URL Shortener is that it's open source! It's licensed under the MIT License so you can do whatever you want with it.

API

Simple URL Shortner has a very simple API you can use to shorten links.
No API key needed.

URL: http://surls.ga/shorten.php
